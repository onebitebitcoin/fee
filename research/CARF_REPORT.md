# CARF (암호화자산 보고 프레임워크) 연구 보고서

> **조사 기준일**: 2026년 3월 16일
> **출처**: OECD 공식 문서, 각 거래소 공식 사이트, 금융위원회/국세청 자료

---

## 1. CARF란?

CARF(Crypto-Asset Reporting Framework)는 OECD가 주도하는 암호화자산 자동 정보교환 프레임워크입니다. 각국 세무당국이 자국민의 해외 암호화자산 거래 정보를 자동으로 교환해 탈세를 방지하는 것이 목적입니다.

## 3. 한국의 CARF 정책

| 항목 | 내용 |
|------|------|
| MCAA 서명 | 2024년 11월 27일 (OECD 글로벌포럼 17차 총회) |
| 서명국 수 | 48개국 동시 서명 |
| 조기 도입국 여부 | 실질적으로 2027년 첫 교환 = 조기 도입국과 동일 일정 |
| 데이터 수집 시작 | **2026년 1월 1일** (이미 시작됨) |
| 국내 보고 기한 | **2027년 4월 말** (거래소 → 국세청) |
| 첫 국제 정보교환 | **2027년 중** |
| 가상자산 과세 시행 | **2027년 1월 1일** (소득세법 2024년 12월 개정) |
| 세율 | 연간 250만 원 초과분에 22% (지방소득세 포함) |
| 관련 법령 | 특금법, 소득세법, 기재부 이행규정 (2025년 행정예고) |

## 7. 핵심 인사이트

1. **한국 5대 원화 거래소는 모두 2027년 CARF 첫 교환 대상** — 2026년 거래부터 해외 과세당국과 공유됨
2. **Gate.io가 한국 공식 서비스 제공 유일 글로벌 거래소** — 케이맨 CARF 2027 적용으로 한국 사용자 데이터 교환 대상
3. **Binance는 케이맨 CARF 2027** — 한국 앱은 차단이나 우회 이용자 존재 시 정보교환 가능
4. **KuCoin/HTX는 CARF 완전 사각지대** — 세금 추적 불가, 규제 리스크 최고
5. **2026년 1월부터 데이터 수집 이미 시작** — 한국 사용자는 현재 모든 거래가 기록되고 있음
6. **가상자산 과세(2027-01) + CARF 첫 교환(2027) 동시 시행** — 국내외 포지션 모두 세금 신고 필요
