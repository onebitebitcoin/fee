# UI Reference Notes

Source image: `design/reference.png`

Observed design traits:
- Full dark terminal-like canvas with almost-black background and subtle panel borders.
- Top control row with four blocks: amount input, target priority select, output type select, and strong blue CTA button.
- Four compact stat cards under controls for mempool backlog and fee priorities.
- Large table section titled `ROUTE OPTIMIZATION ANALYSIS` with path/provider/fee/cost/status columns.
- Bottom bar chart section titled `FEE RATE VELOCITY (24H)`.
- Typography is uppercase, compact, high-contrast, with muted gray labels and blue highlights.
- Table status pills use green/gray/red accents.
